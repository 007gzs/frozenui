# FrozenUI 与 JMUI 共建UI

## 目录结构

1. 以下为与 JMUI 源码编写相关目录

- sass/
    + 存放 .scss 源码

- css/
    + frozen.css

- js/
    + frozen.js

- lib/
    + 存放第三方依赖类库，如 zeptojs

- doc/
    + 代码规范

- jsdoc/
    + jsdoc生成的api文档

2. 以下为网站文档编写相关目录

- demo/
    + 示例, 从 docs/ 自动生成

- site-config.js
    + 生成网站文档所需配置

- Gruntfile.js
    + grunt 配置文件
